# 📱 Dem1chVPN — Подключение на Android

## Что нужно

- **ОС:** Android 5.0 и выше
- **Клиент:** [v2rayNG](https://play.google.com/store/apps/details?id=com.v2ray.ang)
- **Личный кабинет Dem1chVPN** (доступен через Telegram-бота)

---

## Пошаговая инструкция

### 1. Установите v2rayNG

**Из Google Play (рекомендуется):**
- [v2rayNG в Google Play](https://play.google.com/store/apps/details?id=com.v2ray.ang)

**Из GitHub (если Google Play недоступен):**
1. Перейдите на [github.com/2dust/v2rayNG/releases](https://github.com/2dust/v2rayNG/releases)
2. Скачайте APK-файл для вашей архитектуры:
   - `v2rayNG_*_arm64-v8a.apk` — для современных устройств (рекомендуется)
   - `v2rayNG_*_universal.apk` — если не знаете архитектуру
3. Установите APK (потребуется разрешение на установку из неизвестных источников).

### 2. Добавьте сервер

**Способ 1 — Подписка (рекомендуется, автообновление):**
1. Откройте **Личный кабинет** через бота 
2. Скопируйте **URL подписки** в разделе "Ваши ссылки"
3. В приложении v2rayNG: `☰` (меню слева) → `Группа подписки` → `+` (добавить)
4. Вставьте скопированный URL в поле "URL" и сохраните (галочка).
5. Вернитесь на главный экран: `☰` → `Обновить подписку`.

**Способ 2 — QR-код (самый простой):**
1. Запросите QR-код у бота (меню управления юзерами).
2. В v2rayNG нажмите `+` (сверху) → `Сканировать QR-код`.
3. Наведите камеру на QR-код или выберите скриншот.

**Способ 3 — Через ссылку VLESS:**
1. Скопируйте прямую `vless://` ссылку из бота.
2. В v2rayNG нажмите `+` → `Импорт из буфера обмена`.

### 3. Настройте маршрутизацию (Раздельный туннель)

1. В v2rayNG: `☰` → `Настройки` → `Предустановленные правила` (Routing settings).
2. Выберите или добавьте логику `Обход трафика для российских сайтов` (или `Bypass LAN and mainland`).
3. Режим Domain Strategy: поставьте `IPIfNonMatch`.

> 💡 Это позволит приложениям банков, Госуслугам и Яндекс-сервисам работать напрямую без VPN, сохраняя полную скорость.

### 4. Подключитесь

1. Выберите появившийся сервер из списка (просто нажмите на него, чтобы он подсветился).
2. Нажмите круглую кнопку `▶` внизу экрана.
3. При первом подключении Android запросит системное разрешение на VPN — нажмите «ОК» или «Разрешить».
4. В статус-баре появится значок ключа 🔑.

### 5. Проверьте

- Откройте YouTube — должен грузиться моментально ✅
- Откройте 2ip.ru — покажет IP вашего VPS (европейский IP) ✅
- Откройте Сбербанк или Яндекс — должен работать напрямую без задержек ✅

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Не подключается | Убедитесь, что любой другой VPN или AdGuard ВЫКЛЮЧЕН. |
| Перерасход батареи | v2rayNG потребляет минимум, основная нагрузка — от самих приложений (YouTube, Instagram). |
| YouTube не открывается | Проверьте, что домен youtube.com идёт через Proxy. По умолчанию все иностранные сайты так и открываются. |
| Нет уведомления в статус-баре | Откройте системные настройки Android → Приложения → v2rayNG → Разрешения на уведомления. |

## Полезные советы

- 🔋 **Энергосбережение:** В настройках телефона добавьте приложение v2rayNG в "исключения оптимизации батареи", чтобы система не убивала VPN в фоне.
- 📡 **Стабильность:** Для более стабильного соединения установите параметр «Провайдер sniffing» = Включен (в настройках v2rayNG).
- 🔄 **Автообновления:** Обязательно используйте "Подписку" вместо QR/VLESS ссылок. Если настройки сервера изменятся, ваше приложение скачает их само!
