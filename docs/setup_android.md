# Dem1chVPN — Android

## Что нужно

- Android 5.0+
- Клиент — зависит от того, какой протокол хотите использовать (см. ниже)
- Личный кабинет Dem1chVPN в боте

---

## Какой клиент выбрать

Подписка содержит две ссылки — VLESS+Reality (TCP/443) и Hysteria2 (UDP/8444). Один клиент оба сразу подхватит не везде:

| Клиент | Движок | VLESS+Reality | Hysteria2 |
|--------|--------|:-------------:|:---------:|
| v2rayNG | Xray-core | да | нет |
| NekoBox for Android | sing-box | да | да |
| Hiddify Next | sing-box | да | да |
| sing-box (официальный APK) | sing-box | да | да |

**Если хочется обоих протоколов в одном приложении — ставьте NekoBox или Hiddify Next.** v2rayNG проигнорирует `hysteria2://` из подписки.

**Если ТСПУ начнёт резать VLESS+TCP** (что и стало причиной добавления второго протокола) — у пользователей v2rayNG останется только Hysteria2-ссылка, которую он не умеет. То есть на Android именно для запаса смысл переходить на NekoBox/Hiddify.

## Вариант A: NekoBox for Android (рекомендуется)

Поддерживает оба протокола, активный сервер переключается одним тапом.

### 1. Установка

- [NekoBox на GitHub Releases](https://github.com/MatsuriDayo/NekoBoxForAndroid/releases) — скачать APK для вашей архитектуры (`arm64-v8a` для большинства устройств).
- Установить APK (разрешить установку из неизвестных источников, если попросит).

В Google Play NekoBox нет — только GitHub или F-Droid.

### 2. Добавить подписку

1. Скопировать URL подписки в Личном кабинете в боте.
2. В NekoBox: правый верхний угол `≡` → «Группы» → `+`.
3. Тип: «Удалённый профиль», вставить URL.
4. Сохранить, выбрать группу, нажать обновить.
5. В списке серверов появятся два — VLESS и Hysteria2.

### 3. Маршрутизация

NekoBox умеет split-tunneling по правилам. Из коробки можно включить «Bypass LAN and mainland China» — российские сети туда не входят, поэтому лучше использовать наш набор правил:

- Настройки → Параметры маршрутизации → Импорт правил.
- Можно вставить URL правил из бота (если поддерживается версией) или настроить вручную: домены `geosite:category-ru` и IP `geoip:ru` → direct.

### 4. Подключиться

1. Выбрать сервер (VLESS либо Hysteria2).
2. Нижняя кнопка подключения.
3. Разрешить VPN.

Если один протокол перестал работать — выбираете в списке другой и переподключаетесь.

## Вариант B: v2rayNG (только VLESS)

Если устраивает только VLESS, и нет желания ставить NekoBox.

### 1. Установка

- [v2rayNG в Google Play](https://play.google.com/store/apps/details?id=com.v2ray.ang)
- Или [APK с GitHub](https://github.com/2dust/v2rayNG/releases) (`v2rayNG_*_arm64-v8a.apk` для большинства устройств).

### 2. Добавить сервер

По подписке:

1. Скопировать URL подписки из бота.
2. `☰` → «Subscription group settings» → `+`.
3. Вставить URL, сохранить.
4. `☰` → «Update subscription».
5. В списке появится один сервер — VLESS (Hysteria2-ссылка из подписки будет проигнорирована).

По QR:

1. Бот: Пользователи → выбрать → QR.
2. v2rayNG: `+` → «Сканировать QR-код».

По прямой ссылке:

1. Скопировать `vless://...` из бота.
2. `+` → «Импорт из буфера».

### 3. Маршрутизация

1. `☰` → «Settings» → «Routing settings».
2. Выбрать или адаптировать «Bypass LAN and mainland».
3. Domain Strategy: `IPIfNonMatch`.

### 4. Подключиться

1. Тапнуть на сервер.
2. Кнопка `▶` снизу.
3. Разрешить VPN.

## Hiddify Next (альтернатива)

Тоже sing-box, поддерживает оба протокола. UI ближе к обычным VPN-приложениям, чем у NekoBox.

- [Google Play](https://play.google.com/store/apps/details?id=app.hiddify.com) или [GitHub](https://github.com/hiddify/hiddify-next/releases).
- Импорт подписки: главный экран → `+` → вставить URL.

## Проверка

- YouTube открывается без замедления.
- 2ip.ru показывает европейский IP вашего VPS.
- Сбер, Госуслуги, Яндекс работают без задержек — значит маршрутизация работает.

## Проблемы

| Симптом | Что попробовать |
|---------|-----------------|
| В v2rayNG в подписке только один сервер | Это норма — v2rayNG не умеет Hysteria2, видит только VLESS. Поставьте NekoBox, если нужен второй протокол |
| Hysteria2 не подключается на NekoBox | Провайдер режет UDP — переключиться на VLESS в списке серверов |
| VLESS не подключается | ТСПУ режет TCP — попробовать Hysteria2 (только NekoBox/Hiddify) |
| Не подключается вообще | Выключить любой другой VPN, AdGuard и прочие фильтры |
| Жрёт батарею | Сам клиент ест мало, основная нагрузка от YouTube/Instagram |
| Нет уведомления в статус-баре | Настройки Android → Приложения → клиент → Уведомления |

## Советы

- Клиент стоит добавить в исключения оптимизации батареи, иначе Android может прибить VPN в фоне.
- Включите Sniffing/Domain Sniffing в настройках — стабильнее работает split-tunneling по доменам.
- Используйте URL подписки, а не отдельные ссылки. При смене ключей на сервере новые приедут сами.
