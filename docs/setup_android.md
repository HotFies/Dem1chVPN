# 📱 Dem1chVPN — Подключение на Android

## Что нужно

- **Клиент:** [v2rayNG](https://github.com/2dust/v2rayNG/releases) (Android 5.0+)
- **Ссылка подключения** или **QR-код** от бота Dem1chVPN

---

## Пошаговая инструкция

### 1. Установите v2rayNG

**Из Google Play (рекомендуется):**
- [v2rayNG в Google Play](https://play.google.com/store/apps/details?id=com.v2ray.ang)

**Из GitHub (если Play недоступен):**
1. Перейдите на [github.com/2dust/v2rayNG/releases](https://github.com/2dust/v2rayNG/releases)
2. Скачайте APK-файл для вашей архитектуры:
   - `v2rayNG_*_arm64-v8a.apk` — для современных устройств (рекомендуется)
   - `v2rayNG_*_universal.apk` — если не знаете архитектуру
3. Установите APK (может потребоваться разрешение на установку из неизвестных источников)

### 2. Добавьте сервер

**Способ 1 — QR-код (самый простой):**
1. Запросите QR-код у бота Dem1chVPN
2. В v2rayNG нажмите `+` → `Сканировать QR-код`
3. Наведите камеру на QR-код

**Способ 2 — Через ссылку:**
1. Скопируйте VLESS-ссылку из бота
2. В v2rayNG нажмите `+` → `Импорт из буфера обмена`

**Способ 3 — Подписка (автообновление):**
1. В v2rayNG: `☰` → `Группа подписки` → `+`
2. Вставьте URL подписки из бота
3. Вернитесь и нажмите `Обновить подписку`

### 3. Настройте маршрутизацию

1. В v2rayNG: `☰` → `Настройки` → `Предустановленные правила`
2. Выберите `Обход трафика для российских сайтов` или `Bypass LAN and mainland`

> 💡 Это позволит Яндексу, ВК, Сберу и другим российским сервисам работать напрямую

### 4. Подключитесь

1. Выберите сервер из списка (тапните на него)
2. Нажмите кнопку `▶` внизу экрана
3. При первом подключении Android запросит разрешение на VPN — нажмите «ОК»
4. В статус-баре появится значок ключа 🔑

### 5. Проверьте

- Откройте YouTube — должен работать ✅
- Откройте 2ip.ru — покажет IP VPS ✅
- Откройте Яндекс — должен работать напрямую ✅

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Не подключается | Убедитесь, что другой VPN выключен |
| Перерасход батареи | v2rayNG потребляет минимум, основная нагрузка — от приложений |
| YouTube не открывается | Проверьте, что домен youtube.com в правилах маршрутизации = PROXY |
| Нет уведомления в статус-баре | Проверьте разрешения приложения в настройках Android |

## Рекомендации

- 🔋 В настройках Android добавьте v2rayNG в исключения оптимизации батареи
- 📡 Для StableConnection: установите «Провайдер sniffing» = Включен
- 🔄 Подписка обновляется автоматически — конфигурация всегда актуальна
