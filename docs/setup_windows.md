# 🖥️ Dem1chVPN — Подключение на Windows

## Что нужно

- **Клиент:** [v2rayN](https://github.com/2dust/v2rayN/releases) (v6.x и выше)
- **Ссылка подключения** или **QR-код** от бота Dem1chVPN

---

## Пошаговая инструкция

### 1. Скачайте v2rayN

1. Перейдите на [github.com/2dust/v2rayN/releases](https://github.com/2dust/v2rayN/releases)
2. Скачайте файл `v2rayN-With-Core.zip` (версия с ядром, ~30 MB)
3. Распакуйте архив в удобную папку (например, `C:\v2rayN`)
4. Запустите `v2rayN.exe`

> ⚠️ **Важно:** при первом запуске Windows может запросить разрешение брандмауэра — нажмите «Разрешить»

### 2. Добавьте сервер

**Способ 1 — Через ссылку (рекомендуется):**
1. Скопируйте VLESS-ссылку из бота (начинается с `vless://...`)
2. В v2rayN нажмите `Серверы` → `Импортировать из буфера обмена`
3. Сервер появится в списке

**Способ 2 — Через подписку (автообновление):**
1. В v2rayN нажмите `Подписки` → `Настройки подписки`
2. Добавьте URL подписки из бота
3. Нажмите `Обновить подписки`
4. Конфигурация будет обновляться автоматически каждые 6 часов

### 3. Настройте маршрутизацию (раздельный туннель)

1. В v2rayN перейдите в `Настройки` → `Настройки маршрутизации`
2. Импортируйте файл `configs/v2rayn_routing.json` из проекта
3. Или выберите встроенный шаблон «Обход трафика Китая» (похожая логика)

> 💡 Раздельная маршрутизация позволяет российским сайтам (Яндекс, ВК, Госуслуги) работать **напрямую**, без потери скорости

### 4. Подключитесь

1. В v2rayN выберите ваш сервер из списка
2. Нажмите кнопку `▶` (Старт) внизу окна
3. В трее появится зелёная иконка v2rayN

### 5. Проверьте подключение

Откройте в браузере:
- [https://youtube.com](https://youtube.com) — должен открыться
- [https://2ip.ru](https://2ip.ru) — покажет IP вашего VPS (не вашего провайдера)

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Не подключается | Проверьте, что антивирус не блокирует v2rayN |
| Медленная скорость | Убедитесь, что маршрутизация настроена (российские сайты — напрямую) |
| v2rayN не запускается | Установите [.NET Desktop Runtime 6.0+](https://dotnet.microsoft.com/download/dotnet/6.0) |
| Нет интернета при VPN | Переключите режим маршрутизации на «Обход LAN и Китая» |
