# Dem1chVPN — Windows

## Что нужно

- Windows 10 или 11
- Личный кабинет Dem1chVPN либо ссылка подключения из бота

---

## Вариант 1: нативный клиент Dem1chVPN

Внутри уже sing-box и автоимпорт. Поддерживает оба протокола (VLESS и Hysteria2) с переключением одной кнопкой.

### 1. Скачать

1. Скачать `Dem1chVPN-Setup.exe` (последнюю версию) с [GitHub Releases](https://github.com/HotFies/Dem1chVPN/releases/latest).
2. Запустить установщик.

Если Windows SmartScreen ругается — «Подробнее» → «Выполнить в любом случае».

### 2. Импорт подписки

Через deeplink:

1. Открыть Личный кабинет в боте (кнопка «Открыть панель»).
2. В разделе «Ваши ссылки» — строка «Импорт подписки (Windows)» со ссылкой `dem1chvpn://import/...`.
3. Кликнуть, разрешить браузеру открыть приложение.
4. Подписка добавится сама.

Вручную:

1. Скопировать URL подписки в боте.
2. Открыть приложение, вставить в поле ввода подписки в настройках.

### 3. Выбор протокола

В подписке всегда обе ссылки — VLESS (TCP/443) и Hysteria2 (UDP/8444). На главном экране в Dashboard есть переключатель.

- По умолчанию активен Hysteria2 — он сейчас стабильнее в РФ под ТСПУ.
- Если на Hysteria2 что-то не идёт (например, провайдер режет UDP) — переключаетесь на VLESS.

После переключения клиент сам перегенерирует sing-box-конфиг и переподнимет туннель.

### 4. Подключиться

Кнопка подключения на главном экране. По дефолту настроена раздельная маршрутизация: российские сайты идут напрямую, заблокированные — через туннель.

## Вариант 2: v2rayN

Для тех, кто хочет ковыряться вручную.

### 1. Скачать

1. [GitHub Releases](https://github.com/2dust/v2rayN/releases).
2. Скачать `v2rayN-windows-64-SelfContained.zip` (внутри уже .NET).
3. Распаковать в удобную папку (например, `C:\v2rayN`).
4. Запустить `v2rayN.exe`, при запросе брандмауэра — «Разрешить».

### 2. Подписка

1. Подписки → Настройки подписки.
2. Добавить, вставить URL подписки из бота, сохранить.
3. Подписки → Обновить подписки.

Из подписки приедут оба сервера — VLESS и Hysteria2. В списке выбираете нужный.

### 3. Маршрутизация

1. Настройки → Настройки маршрутизации.
2. Выбрать или создать правило «Bypass LAN and mainland» (адаптировать под РФ).
3. Российские сайты пойдут напрямую.

### 4. Подключиться

1. Выбрать сервер в списке (один клик).
2. Enter или правой кнопкой → «Установить как активный».
3. В трее изменится иконка. Включить «Set system proxy».

## Проверка

- [youtube.com](https://youtube.com) — должен открыться без замедления.
- [2ip.ru](https://2ip.ru) — должен показать IP вашего VPS.

## Проблемы

| Симптом | Что попробовать |
|---------|-----------------|
| Не подключается | Антивирус может блокировать Dem1chVPN/v2rayN. Добавить в исключения |
| Hysteria2 не работает | Провайдер мог зарезать UDP/8444. Переключиться на VLESS в Dashboard |
| VLESS не работает | ТСПУ начал резать TCP — переключиться на Hysteria2 |
| Медленно | Проверить, что маршруты подгрузились (российские сайты должны открываться напрямую) |
| v2rayN не запускается | Поставить [.NET Desktop Runtime 6.0+](https://dotnet.microsoft.com/download/dotnet/6.0), если не SelfContained сборка |
| Нет интернета при VPN | Перезапустить приложение. В v2rayN: Clear system proxy → Set system proxy |
