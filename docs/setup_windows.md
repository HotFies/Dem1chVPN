# 🖥️ Dem1chVPN — Подключение на Windows

## Что нужно

- **ОС:** Windows 10 / 11
- **Личный кабинет Dem1chVPN** или ссылка подключения (из Telegram-бота)

---

## Вариант 1 (Рекомендуется) — Нативный клиент Dem1chVPN

Мы разработали собственный клиент для Windows, который максимально прост в установке и имеет встроенное ядро и автоимпорт настроек.

### 1. Скачайте приложение

1. Перейдите на [GitHub Releases](https://github.com/HotFies/Dem1chVPN/releases/download/demichvpn-win-v.1.0.0/Dem1chVPN-1.0.0-Setup.exe) и скачайте `Dem1chVPN-Setup.exe`.
2. Запустите загруженный установщик и следуйте инструкциям на экране.
   *(Если Windows SmartScreen выдаст предупреждение, нажмите "Подробнее" → "Выполнить в любом случае").*

### 2. Импортируйте подписку

**Через Deeplink (в 1 клик):**
1. Откройте **Личный кабинет** через Telegram-бота (`📱 Открыть панель`).
2. В разделе *Ваши ссылки* найдите строку **«Импорт подписки (Windows)»** с ссылкой `dem1chvpn://sub/...`.
3. Нажмите на неё — браузер предложит открыть приложение Dem1chVPN.
4. Разрешите открытие — подписка автоматически добавится в приложение.

**Вручную:**
1. В личном кабинете или меню бота скопируйте **URL подписки**.
2. Откройте приложение Dem1chVPN.
3. Вставьте скопированную ссылку в поле ввода подписки в настройках.

### 3. Подключитесь

1. На главном экране приложения просто нажмите на кнопку старта/подключения.
2. Готово! По умолчанию настроена раздельная маршрутизация (Российские сайты идут напрямую без потери скорости, а заблокированные — через туннель).

## Вариант 2 (Альтернатива) — v2rayN

Если вы опытный пользователь и вам нужны расширенные настройки, вы можете использовать v2rayN.

### 1. Скачайте v2rayN

1. Перейдите на [github.com/2dust/v2rayN/releases](https://github.com/2dust/v2rayN/releases).
2. Скачайте `v2rayN-windows-64-SelfContained.zip` (версия, которая уже содержит .NET).
3. Распакуйте архив в удобную папку (например, `C:\v2rayN`).
4. Запустите `v2rayN.exe`. *(При запросе брандмауэра нажмите «Разрешить»).*

### 2. Добавьте сервер

**Через подписку (рекомендуется — автообновление каждые 6 часов):**
1. В v2rayN выберите `Подписки` (Subscription) → `Настройки подписки` (Subscription settings).
2. Нажмите `Добавить` (Add), вставьте **URL подписки** из бота в поле URL. Сохраните.
3. В главном окне нажмите `Подписки` → `Обновить подписки` (Update subscription).

### 3. Настройте маршрутизацию (раздельный туннель)

1. В v2rayN перейдите в `Настройки` (Settings) → `Настройки маршрутизации` (Routing setting).
2. Выберите или создайте правило для «Обход трафика Китая»/России (Bypass LAN and mainland).
3. Это позволит российским сайтам работать напрямую минуя VPN.

### 4. Подключитесь

1. В списке серверов выберите скачанный сервер (нажмите на него).
2. Нажмите Enter, либо кликните правой кнопкой мыши → "Установить как активный сервер".
3. В системном трее иконка v2rayN должна сменить цвет (по умолчанию нужно установить System Proxy в "Set system proxy").

## 🧪 Проверка подключения

Откройте в браузере:
- [https://youtube.com](https://youtube.com) — должен открыться быстро и без ограничений.
- [https://2ip.ru](https://2ip.ru) — должен показать IP вашего VPS сервера в Европе.

## ❗️ Решение проблем

| Проблема | Решение |
|----------|---------|
| Не подключается | Проверьте, что антивирус не блокирует Dem1chVPN или v2rayN. |
| Медленная скорость | Убедитесь, что маршрутизация скачалась корректно (российские сайты должны открываться напрямую). |
| v2rayN не запускается | Установите [.NET Desktop Runtime 6.0+](https://dotnet.microsoft.com/download/dotnet/6.0) (если используете не SelfContained версию). |
| Нет интернета при VPN | Перезапустите приложение. В v2rayN проверьте режим `Routing setting` (Clear system proxy, затем Set system proxy). |
