# 🍎 Dem1chVPN — Подключение на iOS (iPhone/iPad)

## Что нужно

- **Клиент:** [Streisand](https://apps.apple.com/app/streisand/id6450534064) (рекомендуется, бесплатно) или [V2Box](https://apps.apple.com/app/v2box-v2ray-client/id6446814690)
- **Ссылка подключения** или **URL подписки** от бота Dem1chVPN
- iOS/iPadOS 15.0 и выше

> ⚠️ **НЕ используйте V2RayTun** — он устаревший, не поддерживает `xtls-rprx-vision` flow, и имеет проблемы с маршрутизацией и стабильностью. Используйте Streisand или V2Box.

---

## Рекомендуемый клиент: Streisand

### 1. Установите Streisand

1. Откройте [Streisand в App Store](https://apps.apple.com/app/streisand/id6450534064)
2. Установите приложение (бесплатно)

> 💡 Альтернативы: [V2Box](https://apps.apple.com/app/v2box-v2ray-client/id6446814690) (бесплатно) или [Shadowrocket](https://apps.apple.com/app/shadowrocket/id932747118) (платно, $2.99)

### 2. Добавьте сервер

**Способ 1 — Подписка (рекомендуется):**
1. Откройте бота Dem1chVPN в Telegram
2. Скопируйте URL подписки (начинается с `https://...`)
3. В Streisand: `Настройки` → `Подписки` → `+`
4. Вставьте URL подписки
5. Нажмите «Обновить»
6. ✅ Конфигурация будет обновляться автоматически каждые 6 часов

**Способ 2 — QR-код:**
1. Запросите QR-код у бота Dem1chVPN
2. В Streisand нажмите `+` → `Сканировать QR`
3. Наведите камеру на QR-код

**Способ 3 — VLESS-ссылка:**
1. Скопируйте VLESS-ссылку из бота в Telegram
2. Откройте Streisand — он автоматически предложит импортировать
3. Или: нажмите `+` → `Из буфера обмена`

### 3. Настройте режим маршрутизации

> 🔑 **Это ключевой шаг для скорости и стабильности!**

Streisand поддерживает несколько режимов работы:

| Режим | Описание | Когда использовать |
|-------|----------|-------------------|
| **Global** | Весь трафик через VPN | Когда нужен доступ ко всему (TikTok, YouTube, NotebookLM) |
| **Rule** | По правилам маршрутизации | Для экономии трафика (банки, Яндекс — напрямую) |
| **Direct** | Весь трафик напрямую | VPN отключён |

**Рекомендация:** начните с **Global**, затем переключитесь на **Rule** для экономии трафика.

Для режима **Rule**:
1. `Настройки` → `Правила маршрутизации`
2. Включите «Обход локальных адресов»
3. Добавьте российские домены в DIRECT:
   - `gosuslugi.ru`, `yandex.ru`, `vk.com`, `sberbank.ru`, `tinkoff.ru`
4. Остальной трафик будет идти через VPN

### 4. Подключитесь

1. Выберите сервер «🛡️ Dem1chVPN» из списка
2. Нажмите кнопку подключения (▶️)
3. При первом подключении iOS запросит разрешение на VPN-профиль
4. Введите код устройства для подтверждения
5. В статус-баре появится значок VPN ✅

### 5. Проверьте работу

- Откройте Safari → youtube.com → должен работать ✅
- Откройте TikTok → видео в ленте и DM должны грузиться ✅
- Откройте notebooklm.google.com → должен открыться ✅
- Откройте 2ip.ru → покажет IP VPS (не российский) ✅
- Откройте sberbank.ru → должен работать (идёт напрямую) ✅

## Альтернативные клиенты

### V2Box

1. Установите [V2Box](https://apps.apple.com/app/v2box-v2ray-client/id6446814690)
2. Перейдите в «Подписки» → вставьте URL
3. Обновите конфигурацию
4. Подключитесь

### Shadowrocket (платный, $2.99)

1. Установите [Shadowrocket](https://apps.apple.com/app/shadowrocket/id932747118)
2. Скопируйте URL подписки или VLESS-ссылку из Telegram
3. Shadowrocket автоматически предложит добавить сервер
4. Включите подключение

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Не подключается | Перезагрузите устройство и попробуйте снова |
| Streisand не видит ссылку | Убедитесь, что ссылка начинается с `vless://` |
| VPN отключается в фоне | «Настройки» → «Основные» → «VPN» → включите «Подключить по запросу» |
| Нет приложения в App Store | Может потребоваться сменить регион App Store |
| TikTok видео не грузятся | Переключите режим на **Global** вместо Rule |
| NotebookLM не открывается | Проверьте что WARP включён на сервере (через бота) |
| Сайты грузятся медленно | Переключите режим на **Rule** (российские сайты пойдут напрямую) |
| Часто обрывается соединение | Проверьте сигнал WiFi/LTE, попробуйте отключить «Экономия батареи» |

## Рекомендации

- 🔄 Используйте **подписку** (а не разовую VLESS-ссылку) для автоматического обновления
- 📱 Включите **«VPN по запросу»** для автоматического переподключения
- 🔋 iOS оптимизирует VPN-подключения — батарея расходуется минимально
- 🌐 Для полного доступа ко всем сервисам используйте режим **Global**
- ⚡ Для скорости на российских сайтах используйте режим **Rule**
