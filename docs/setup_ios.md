# Dem1chVPN — iOS (iPhone, iPad)

## Что нужно

- iOS / iPadOS 15.0+
- Клиент [V2RayTun](https://apps.apple.com/app/v2raytun/id6476628951) (рекомендуется)
- Личный кабинет Dem1chVPN в боте

Если V2RayTun недоступен в вашем App Store (в РФ его удалили), смените регион Apple ID на США, Турцию или любую другую страну. Это бесплатно и быстро.

---

## V2RayTun (импорт в один клик)

V2RayTun интегрирован с Личным кабинетом и принимает подписку и маршруты по deeplink.

### 1. Установить

1. [V2RayTun в App Store](https://apps.apple.com/app/v2raytun/id6476628951).
2. Поставить (бесплатно).

### 2. Подписка

1. Открыть Личный кабинет в боте («Открыть панель»).
2. В разделе «Ваши ссылки» — кнопка «Импорт подписки из Telegram (V2RayTun)» (ссылка `v2raytun://import/...`).
3. Нажать, разрешить открытие приложения.
4. Конфиг откроется в V2RayTun и добавится в список серверов.

В подписке два сервера — VLESS (TCP/443) и Hysteria2 (UDP/8444). В V2RayTun они появятся как два пункта. Выбираете тот, что работает у вашего провайдера; обычно Hysteria2 устойчивее под ТСПУ, но если оператор режет UDP — берёте VLESS.

### 3. Маршруты (раздельный туннель)

Чтобы Сбер и Госуслуги работали без VPN, а YouTube — через VPN:

1. Вернуться в Личный кабинет.
2. Под ссылкой подписки — кнопка «Импорт маршрутов» (`v2raytun://import_route/...`).
3. Разрешить открыть в V2RayTun.

### 4. Подключиться

1. На главном экране V2RayTun потянуть список вниз — обновится ping.
2. Выбрать «Dem1chVPN».
3. Нажать ▶.
4. При первом подключении iOS попросит установить VPN-профиль — разрешить, ввести код-пароль iPhone.
5. В статус-баре появится значок VPN.

## Альтернативные клиенты

Подписку нужно будет добавить руками.

### Streisand

1. [Streisand в App Store](https://apps.apple.com/app/streisand/id6450534064), бесплатно.
2. Скопировать URL подписки в Личном кабинете.
3. Настройки → Подписки → `+`.
4. Вставить ссылку, назвать «Dem1chVPN», включить автообновление.
5. На главном экране выбрать сервер, нажать кнопку подключения.
6. Настройки → Правила маршрутизации, чтобы обойти российские сайты (режим Rule).

### V2Box

1. [V2Box в App Store](https://apps.apple.com/app/v2box-v2ray-client/id6446814690), бесплатно.
2. Скопировать URL подписки.
3. Configs → `+` → Add Subscription.
4. Вставить URL, сохранить.
5. Выбрать сервер, Slide to Connect.

### Shadowrocket

1. [Shadowrocket](https://apps.apple.com/app/shadowrocket/id932747118), около $2.99.
2. Скопировать URL подписки из бота.
3. Открыть Shadowrocket — сам предложит добавить сервер из буфера.
4. Включить переключатель сверху.

## Проблемы

| Симптом | Что попробовать |
|---------|-----------------|
| Deeplink из Telegram не открывается | Telegram WebView иногда блокирует кастомные схемы. Скопировать ссылку и открыть в Safari, либо вставить URL подписки руками в настройках клиента |
| VPN отключается в фоне | Настройки iOS → Основные → VPN → (i) рядом с профилем → включить «Подключать по запросу» |
| Hysteria2 не работает | Провайдер режет UDP. В V2RayTun выбрать VLESS-сервер из той же подписки |
| VLESS не работает | ТСПУ режет TCP. Переключиться на Hysteria2 |
| Сайты грузятся медленно | Проверить пинг до серверов. Если совсем ничего — переключить Wi-Fi на LTE или обновить подписку в приложении |
| Приложения банков не работают | Проверить, что маршруты подгружены — они отправляют трафик банков мимо туннеля |

## Советы

- Подписка обновляется автоматически примерно раз в 6 часов. При смене ключей на сервере новые приедут на телефон сами.
- На VLESS+Reality батарея почти не садится. Hysteria2 чуть прожорливее, но разница незаметна на обычном использовании.
