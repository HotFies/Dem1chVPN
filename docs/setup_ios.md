# 🍎 XShield — Подключение на iOS (iPhone/iPad)

## Что нужно

- **Клиент:** [FoXray](https://apps.apple.com/app/foxray/id6448898396) или [Streisand](https://apps.apple.com/app/streisand/id6450534064)
- **Ссылка подключения** или **QR-код** от бота XShield
- iOS/iPadOS 15.0 и выше

---

## Пошаговая инструкция (FoXray)

### 1. Установите FoXray

1. Откройте [FoXray в App Store](https://apps.apple.com/app/foxray/id6448898396)
2. Установите приложение (бесплатно)

> 💡 Если FoXray недоступен в вашем регионе, попробуйте [Streisand](https://apps.apple.com/app/streisand/id6450534064)

### 2. Добавьте сервер

**Способ 1 — QR-код:**
1. Запросите QR-код у бота XShield
2. В FoXray нажмите `+` → `Сканировать QR`
3. Наведите камеру на QR-код

**Способ 2 — Через ссылку:**
1. Скопируйте VLESS-ссылку из бота в Telegram
2. Откройте FoXray — он автоматически предложит импортировать
3. Или: нажмите `+` → `Из буфера обмена`

**Способ 3 — Подписка (рекомендуется):**
1. В FoXray: `Настройки` → `Подписки` → `+`
2. Вставьте URL подписки из бота
3. Нажмите «Обновить»
4. Конфигурация будет обновляться автоматически

### 3. Настройте маршрутизацию

FoXray поддерживает гибкую маршрутизацию:
1. `Настройки` → `Правила маршрутизации`
2. Включите «Обход локальных адресов»
3. Добавьте российские домены в DIRECT (gosuslugi.ru, yandex.ru, vk.com)

### 4. Подключитесь

1. Выберите сервер из списка
2. Нажмите кнопку подключения
3. При первом подключении iOS запросит разрешение на VPN-профиль
4. Введите код устройства для подтверждения
5. В статус-баре появится значок VPN

### 5. Проверьте

- Откройте Safari → youtube.com → должен работать ✅
- Откройте 2ip.ru → покажет IP VPS ✅

## Альтернатива: Streisand

Если FoXray недоступен:

1. Установите [Streisand](https://apps.apple.com/app/streisand/id6450534064)
2. Скопируйте VLESS-ссылку из Telegram
3. Streisand автоматически предложит добавить сервер
4. Включите подключение

---

## Решение проблем

| Проблема | Решение |
|----------|---------|
| Не подключается | Перезагрузите устройство и попробуйте снова |
| FoXray не видит ссылку | Убедитесь, что ссылка начинается с `vless://` |
| VPN отключается в фоне | Проверьте «Настройки» → «Основные» → «VPN» → «Подключить по запросу» |
| Нет приложения в App Store | Может потребоваться сменить регион App Store |

## Рекомендации

- 🔄 Используйте подписку для автоматического обновления конфигурации
- 📱 Включите «VPN по запросу» для автоматического переподключения
- 🔋 iOS оптимизирует VPN-подключения — батарея расходуется минимально
